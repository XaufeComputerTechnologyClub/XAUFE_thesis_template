\officialDemand{
  \textbf{参考文献}
  \begin{officialDemandText}
    参考文献是毕业论文不可缺少的组成部分，
    它反映毕业论文的取材来源、材料的广博程度和材料的可靠程度，也是作者对他人知识成果的承认和尊重。
    参考文献应有权威性，要注意引用最新的文献。
    本科毕业论文（设计）的参考文献一般应不少于10篇，
    其中，
    外文文献不少于2篇，应以近5年的文献为主。
    参考文献按顺序附于文末。顺序号为[1][2][3]……。
  \end{officialDemandText}
}
